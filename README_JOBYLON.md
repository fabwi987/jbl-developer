## Getting up and running

Easiest way to get up and running is to use the provided Docker configuration:

1. git clone
2. cd jbl-developer
3. docker build -t jbl-developer .
4. docker run -p 4567:4567 jbl-developer
5. Go to http://127.0.0.1:4567
